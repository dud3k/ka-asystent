🚀 Zdalne skanowanie

Uruchamia scraper + Gemini na PC zdalnie przez Gist. PC musi mieć uruchomiony node watch.js.

📋 Krok 1: Eksport danych

Kopiuje pełny stan aplikacji (z wiadomościami) do schowka. Wklej go do Claude jako kontekst.

📋 Krok 2: Import skanu bezpośredniego

Tworzy szkielety rekordów z ka-changes-latest.json bez udziału Gemini. Wiadomości bez tłumaczenia — Gemini dorzuci je w kroku 3.

📥 Krok 3: Import patch Gemini

Wgraj plik JSON lub wklej patch wygenerowany przez Gemini.
